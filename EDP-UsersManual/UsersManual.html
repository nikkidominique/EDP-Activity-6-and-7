LOGIN TO THE SYSTEM

A user must login to the system based on their account records in the database in order to access the Employee Management System. If the entered credentials are correct, the user will be directed to the Main Menu screen.

However if the user doesn't have a record in the database, or if the entered credentials are incorrect, a pop-up window will appear and the user won't be able to log in.

MAIN MENU

After successfully logging in, the user will be directed in the Main Menu screen, where he/she can locate the database. The Main Menu contains all functions of the system and the user could choose here the function that he/she would like to perform.

VIEW EMPLOYEE RECORDS

This is the View Employee Records Screen. The functions that could be done here are shown in the screenshots above, when the user click 'View Employees' in the Main Menu. Aside from viewing the employee records, the user are also able to Locate Data in file explorer, Print Data with Microsoft Excel which can be used for reports generation, and Export Data as a CSV file.

As shown in View Employee Records Screen, there is also a button 'Employee Archive', where a user could view the employee archives, as well as Print and Locate a data.

ADD EMPLOYEE RECORD

This is the View Employee Records Screen. This screen will be shown if the user click the 'Add Employee' in the Main Menu. This function will enable a user to add or register a new employee data.

UPDATE / DELETE EMPLOYEE RECORDS

This is the Update / Delete Employee Records Screen. This screen will be shown if the user click the 'Edit/Delete Employee' in the Main Menu. This function will enable a user to update the records of an employee, or delete an employee's record.

VIEW WORK ASSIGNMENT RECORDS

This is the View Work Assignment Records Screen. This screen will be shown if the user click the 'View Work Assignments' in the Main Menu. This function will enable a user to view the necessary information regarding their work assignments as show in the screenshot above.

BACKUP DATABASE

If the user wants to have a backup file of the database, he/she could click the 'Backup Database' in the Main Menu. This function enables a user to save a backup file with his/her chosen valid file name and save it as .sql file.
